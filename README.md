# Projeto CRUD Users

API feita com NODEJS para gerenciamento de usuários (nome e e-mail).

## Rotas

- `POST /users`: cria um usuário
- `GET /users`: lista todos os usuários
- `GET /users/:id`: obtém um usuário por ID
- `DELETE /users/:id`: remove um usuário

## Como rodar

```bash
npm install
npm run start:dev
